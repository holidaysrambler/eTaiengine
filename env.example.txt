# eT's AI Builder - Environment Variables
# ============================================

# NOTE: When running on Puter, NO API KEYS ARE NEEDED!
# Puter provides keyless access to Claude, Gemini, Mistral, and Perplexity models.
# The app will automatically detect if running on Puter and use keyless mode.

# These variables are ONLY needed if:
# 1. You're running outside of Puter environment
# 2. You want to use your own API keys instead of Puter's free tier

# Puter API (Optional - auto-detected when running on Puter)
# PUTER_API_KEY=your_puter_api_key_here

# Alternative AI Providers (Optional - only if not using Puter)
# OPENAI_API_KEY=your_openai_api_key_here
# ANTHROPIC_API_KEY=your_anthropic_api_key_here
# GOOGLE_API_KEY=your_google_api_key_here

# Qwen API (Optional - for server-side generation)
# QWEN_API_KEY=your_qwen_api_key_here
# QWEN_API_URL=https://dashscope.aliyuncs.com/api/v1/services/aigc/text-generation/generation

# ============================================
# How it works:
# ============================================
# 1. App detects Puter environment automatically
# 2. If on Puter: Uses keyless puter.ai.chat() API (FREE)
# 3. If not on Puter: Falls back to mock generation or uses API keys above
#
# Supported Models (All FREE on Puter):
# - Claude: sonnet-4-5, opus-4-1, sonnet-4
# - Gemini: 2.5-flash, 2.5-pro, 2.0-flash
# - Mistral: large, medium, codestral
# - Perplexity: sonar, sonar-pro, sonar-reasoning
